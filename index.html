Projects

FacePay is a web application developed in a team of three at BattleHack Toronto 2015. FacePay is a new method of processing online payments. By utilizing your existing webcam on your computer, we simply take a snapshot of you and we will fetch your credit card information to complete the transaction! It is as simple as a couple button clicks. We implemented the front end using JavaScript and CSS, and the back end using PHP and Java. We used the SkyBiometry API to perform facial recognition and detection, and the Braintree API to process online payments.
